Como se carga JavaScript

  • Dentro y al final del head con una etiqueta script y crear todo el codigo js dentro (no recomendable porque ensucia el código html).
  • Dentro y al final del head con una etiqueta script añadir el atributo src y cargar la ruta del archivo JS creado anteriormente 'src="archivoJs" (no recomendable porque el script hace referencia al Dom y el Html carga despues) '
  • Como las dos opciones anteriores pero al final del body , ya que el html necesita cargar antes el script. Siendo lamejor forma crear un archivo JS y cargarlo a traves de etiqueta script src="archivo.js".

Tipos de datos

  • undefined. Dato no encontrado
  • null. No existe
  • string o cadena de caracteres. (Entre comillas)

    • var name = "Luis Alfonso"

  • number.

    var number = 3

    • Entero 45
    • Racional 4.66
    • Negativo -8
  • Boolean. true / false

Variables y constantes

Son un tipo de contenedor donde podemos guardar datos y en algunos casos modificarlos

  • var. Como su nombre indica es un contenedor de datos variables. Al no ser un lenguaje fuertemente tipado el tipo de dato puede variar.
    • var number = 5;
    • console.log(number) // 5
    • number = "Cambio a texto"
    • console.log(number) // "Cambio a texto"
  • const. Dato constante, no se puede modificar.
    • const number = 5;
    • console.log(number) // 5
    • number = "Cambio a texto"
    • Error se puede asignar otro valor a una constate
